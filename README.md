# Sorteio Numérico

Aplicação desenvolvida em Python com Flet para realizar sorteios numéricos de forma visual.

## Funcionalidades

- Sorteio de números dentro de um intervalo definido
- Exibição do número sorteado
- Histórico dos números sorteados
- Exportação dos resultados em CSV
- Salvamento automático em TXT

## Tecnologias

- Python
- Flet

## Como executar

```bash
pip install -r requirements.txt
python main.py
